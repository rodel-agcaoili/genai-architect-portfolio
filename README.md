# AWS Generative AI & MLSecOps Portfolio

This repository demonstrates enterprise-grade Generative AI architectures on AWS, 
fully automated via Terraform.

## Key Projects
1. **Secure RAG Pipeline:** Multi-source data ingestion with PII redaction and OpenSearch.
2. **Autonomous Agents:** Bedrock Agents with Cedar-based policy enforcement.
3. **AI Governance Gateway:** Token-aware API management and cost controls.

## Tech Stack
- **IaC:** Terraform
- **AI/ML:** Amazon Bedrock (Claude 3.5, Nova), Amazon Rekognition
- **Orchestration:** Step Functions, Lambda, Bedrock AgentCore
- **DevOps/MLOps:** GitHub Actions, Bedrock Evaluations
